# 解説
## create-react-app
- `create-react-app`で環境構築を省略している
- 公式ドキュメント : [https://create-react-app.dev/](https://create-react-app.dev/)
- 公式サイトには、環境構築後の次のような案内もある
    - エディター設定
    - スタイル設定(CSS/SASS/フォント/画像...)
    - コードの分割方法(モジュール化)
    - 静的データの使用方法
    - Bootstrapの導入方法
    - ルーティングの導入方法
    - TypeScriptの導入方法
    - 自動テストの導入
    - パフォーマンス測定方法
    - APIのプロキシリクエスト
    - GitHub Pagesを用いたサイト公開方法
    - AJAX通信方法(FetchAPIやaxiosの案内)
